# m-yabo-notes

一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 资源的集中存放与版本管理空间。

## 项目简介

本仓库旨在收集、整理并发布一系列独立的 HTML 页面。这些页面可能包含笔记、工具界面、演示文稿或其他形式的静态内容。每个页面均为自包含的 HTML 文件，不依赖外部服务或特定后端。

## 目录说明

```
m-yabo-notes/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面所需的静态资源（图片、样式、脚本等）
├── index.html      # 仓库首页（可选）
└── README.md       # 本文件
```

- `pages/` 目录下每个子文件夹或文件代表一个独立项目或页面。
- `assets/` 目录存放页面共享的资源文件，避免重复引用。

## 页面归档说明

- 所有页面均为静态 HTML，可直接通过浏览器打开。
- 页面内容保持原样归档，不做功能性修改。
- 版本历史通过 Git 提交记录管理，便于回溯与对比。
- 页面标题、描述及元数据应尽量完整，以便于识别和索引。

## 维护说明

- 欢迎通过 Pull Request 提交新的页面或修正现有页面。
- 提交前请确保页面为独立、自包含的 HTML，不包含恶意代码或外部依赖。
- 如页面需要资源文件，请一并放置在 `assets/` 对应目录下。
- 建议在提交信息中简要描述页面用途或变更内容。

## 使用方式

1. 克隆仓库到本地。
2. 直接打开 `pages/` 下的 HTML 文件即可查看。
3. 也可通过 GitHub Pages 或其他静态托管服务发布后在线访问。
